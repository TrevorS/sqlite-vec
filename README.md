## Original Project

All credit for sqlite-vec goes to Alex Garcia ([@asg017](https://github.com/asg017)). sqlite-vec is an extremely small, "fast enough" vector search SQLite extension that runs anywhere.

**Original repository:** https://github.com/asg017/sqlite-vec
